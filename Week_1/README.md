## Description

This exercise demonstrates metadata creation using DataCite XML and schema.org JSON-LD for the Dengue Diseases Dataset.

The dataset contains medical parameters such as age, gender, hemoglobin level, WBC count, RBC count, platelet count, platelet distribution width, and dengue diagnosis label used for disease prediction and machine learning classification.
